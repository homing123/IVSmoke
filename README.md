# IVSmoke

  **Interactive Volumetric Smoke** - A real-time voxel-based volumetric smoke plugin for Unreal Engine 5.7

  ![Unreal Engine](https://img.shields.io/badge/Unreal%20Engine-5.7-blue)
  ![Platform](https://img.shields.io/badge/Platform-Win64-lightgrey)

  ## Overview

  IVSmoke is a GPU-accelerated volumetric smoke solution designed for tactical shooters and action games. Create realistic smoke grenades that dynamically react to bullets, explosions, and moving objects.

  ## Features

  - **Voxel-based Volumetric Smoke** - Real-time ray marching with configurable quality levels
  - **Bullet Penetration** - Projectiles create cylindrical holes through smoke
  - **Explosion Interaction** - Grenades push smoke outward with customizable distortion curves
  - **Dynamic Object Tracking** - Vehicles and characters carve paths through smoke
  - **Network Replication** - Built-in multiplayer support with server-authoritative simulation
  - **Data-Driven Presets** - Reusable Smoke Presets and Hole Presets for easy configuration
  - **Project Settings Integration** - Global quality control via Editor preferences

  ## Requirements

  - Unreal Engine 5.7
  - Windows 10 or later
  - DirectX 11.3+ / Shader Model 5
  - Win64 platform
